# Geldkompass

A personal finance visualizer for German investors. Track accounts, project wealth growth with compound interest, model what-if scenarios, and see your monthly cash flow — with German tax details (Kapitalertragsteuer, Freistellungsauftrag, Rürup) built in.

Built with SvelteKit, Tailwind, shadcn-svelte.

## Dev

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
